Pill badge for statuses (pago/pendente, receita/despesa, stock levels, sale stages).

```jsx
<Badge tone="success">Pago</Badge>
<Badge tone="warning" dot>Pendente</Badge>
<Badge tone="danger">Sem estoque</Badge>
```

Tones: neutral, primary, success, warning, danger — each maps to a semantic bg/text pair. `dot` adds a leading status dot. Always 9999px radius.
